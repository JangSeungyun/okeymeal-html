# OkeyMeal 문서 허브 (OkeyMeal Document Hub)

본 레포지토리는 **OkeyMeal 프로젝트**의 시각화 웹 문서(HTML, CSS, JS 등)를 호스팅하기 위한 전용 저장소입니다.
GitHub Pages를 통해 프로젝트의 기획, 분석, 설계 관련 문서들을 웹 브라우저에서 편리하게 열람할 수 있도록 제공합니다.

## 🔗 접속 링크
- **문서 허브 웹사이트**: [https://jangseungyun.github.io/okeymeal-html/](https://jangseungyun.github.io/okeymeal-html/)

## 📄 주요 내용
본 레포지토리는 프로젝트 산출물 중 HTML로 시각화된 문서들을 포함하고 있습니다.
- **01. Analysis Strategy**: 과제 분석, 타겟 페르소나 등
- **02. Planning Design**: 요구사항 정의서, 서비스 구조도 등
- 프로젝트 가이드라인 및 참조 문서

## 📌 참고 사항
- 본 레포지토리는 문서를 시각적으로 보여주기 위한 웹 뷰어(Web Viewer) 역할을 수행합니다.
- 실제 프로젝트 개발 관련 메인 소스 코드 및 변경 이력은 [OkeyMeal 메인 레포지토리](https://github.com/JangSeungyun/okeymeal)에서 관리됩니다.
