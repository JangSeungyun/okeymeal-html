# 👥 타겟 페르소나 분석서

OkeyMeal의 서비스 가치를 극대화할 수 있는 핵심 페르소나(사용자 3명 + 점주 1명)를 정의합니다.

> 📌 **단일 진실 공급원(SSOT) 정책**
> - 본 문서는 페르소나의 **서사·통증 포인트·솔루션 시나리오**의 정본입니다.
> - **FR ID 매칭표의 정본은** [`01.Ideation/Requirements.md`](../01.Ideation/Requirements.md) §3.1입니다.
> - 페르소나 추가/제거/이름·식이 조건 변경 시 **Requirements §3.1을 동일 커밋에서 갱신**해야 합니다.
> - 본 문서의 각 페르소나 표 마지막 행 "매칭 FR ID"는 Requirements §3.1과 양방향 동기화되어야 합니다.

## 4. 페르소나 D: 디지털 전환이 필요한 소상공인 점주 (Owner - Local Business)

| 항목 | 상세 내용 |
|---|---|
| **이름/나이** | 고순자 (Ko Soon-ja, 61세) |
| **국적/직업** | 한국 / 제주 흑돼지 구이 식당 운영 (22년 경력) |
| **라이프스타일** | 새벽 시장 직접 방문, 신선 재료 고집. 단골 내국인 손님이 많으나 최근 외국인 관광객 유입 증가로 혼란스러움. |
| **스마트폰 활용** | 기본 통화/카카오톡/네이버 지도 사용 가능. 별도 앱 설치·가입에 강한 거부감. |
| **식이 관련 현황** | 메뉴별 정확한 성분 목록 없음. 조리 경험으로 암묵적으로 알고 있으나 외국어로 설명 불가. 돼지기름(라드) 사용 여부, 간장 종류 등을 질문받으면 당황함. |
| **통증 포인트 (Pain Points)** | ① 외국인이 "Is this Halal?" 등으로 물어보면 뭘 묻는지 몰라 그냥 고개를 끄덕이는 경우가 있음. ② 영어 메뉴판 제작 비용(30만원 이상)이 부담. ③ 외국인 손님이 음식 먹고 불쾌한 반응을 보이면 이유를 알 수 없어 불안함. ④ 새로운 앱을 배울 시간과 의지가 없음. |
| **OkeyMeal 솔루션** | **① QR 키트:** 테이블에 스눙스터 QR 스티커를 붙여두기만 하면 됨 — 설치·가입 불필요. **② 한국어 알림 수신:** 외국인 손님이 QR을 스캔하면 카카오톡과 유사한 웹 알림으로 "이 손님은 돼지고기를 못 드십니다. 주방 주의사항: 라드 제거"처럼 한국어로 표시. **③ AI 대체 레시피:** "액젓 → 간장으로 대체 가능합니다" 같이 실행 가능한 한국어 제안 즉시 제공. |
| **온보딩 시나리오** | 제주관광공사 소상공인 지원 프로그램 or 지역 상인회를 통해 QR 키트 오프라인 배포 → 스티커 부착만으로 즉시 서비스 시작 → 점진적으로 메뉴 등록 유도. |
| **매칭 FR ID / NFR** | **OWN-2.1**(실시간 안심 수신함), **OWN-2.2**(원터치 안심 응답), **OWN-2.3**(QR 키트 PDF), **OWN-2.4**(AI 대체 레시피 제안), **OWN-1.1**(스마트 메뉴판 — 점진적 등록), **INF-05**(FCM 단일 채널 — 카카오톡 유사 웹 푸시), **NFR-A11Y-04**(폰트·터치 — 61세 시인성), **CON-08**(점주 콘솔 `owner.okeymeal.kr` 별도 서브도메인) |

## 📊 페르소나별 서비스 활용 기대효과

1.  **신뢰도 기반의 자유로운 이동:** 식이 제약 때문에 특정 지역(예: 할랄 식당 밀집 구역)에만 머물지 않고 전국 어디든 안심하고 여행 가능.
2.  **소통 스트레스 감소:** 언어 장벽으로 인한 주문 실수 및 건강 사고 예방.
3.  **미식 경험의 확장:** 단순히 '먹을 수 있는 것'을 찾는 여행에서 '맛있는 것을 안전하게 즐기는' 여행으로 전환.
