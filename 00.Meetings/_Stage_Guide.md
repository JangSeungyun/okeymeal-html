# 00.Meetings (회의록) 가이드

프로젝트 전반에 걸친 정기 회의, 스프린트 기획, 그리고 단계별(Phase) 회고를 기록하는 공간입니다.

## 📄 작성 필수 문서 지정

1.  **`YYYY-MM-DD-meeting-title.md` (일반 회의록)**
    *   주간 회의나 특정 안건에 대한 논의 기록.
2.  **`YYYY-MM-DD-mN-retrospective-XX.md` (마일스톤 회고록)**
    *   각 Stage(분석, 기획, 개발 등)가 끝날 때 진행하는 KPT 회고 및 다음 단계 Go/No-Go 결정.

---

## 📐 문서별 맞춤 템플릿 규칙

### 1. 일반 회의록
```markdown
## 회의 개요
- **일시**: YYYY-MM-DD HH:MM
- **참석자**: 
- **안건**: 

## 논의 사항
- [논의 내용 상세]

## Action Items
- [ ] [할 일] - 담당자 (기한)
```

### 2. 마일스톤 회고록
```markdown
## KPT 회고
- **Keep**: 좋았던 점, 유지할 점
- **Problem**: 아쉬웠던 점, 문제점
- **Try**: 다음 단계에서 시도할 점

## M(N) 게이트 결정
- **결정**: [Go / Conditional Go / No-Go]
- **사유**: [결정 사유 상세]
```
